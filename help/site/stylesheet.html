Stylesheets

New stylesheet

This dialog is accessible through menu File ▸ Styles ▸ Stylesheet ▸ New.

Give the name of the stylesheet that will be created, opened, and attached to the project.

Depending on the type of stylesheet that was selected, more or less styles will be contained in this new stylesheet.

A basic stylesheet contains only a handful of styles, and the full stylesheet contains them all.
